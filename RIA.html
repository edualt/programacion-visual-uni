Aplicaciones tipo RIA

Rich Internet applications (o RIA) es el término según el cual se define a las aplicaciones web que se comportan del mismo modo que las tradicionales aplicaciones de escritorio, pero con la capacidad de ser ejecutadas en cualquier navegador web. A diferencia de las páginas web clásicas, estas aplicaciones están orientadas a realizar tareas complejas que tradicionalmente habían sido exclusivas de los programas de escritorio, aportando una gran flexibilidad y potencia al usuario final. Para el desarrollo de aplicaciones RIA es necesario combinar una serie de tecnologías complementarias a nivel de cliente y servidor.

Se instalan en el equipo del usuario.

Tecnologías cliente

Las aplicaciones RIA permiten una gran flexibilidad respecto a qué tecnologías usar en la capa de servidor. Es por ello que cualquier entorno puede ser usado sin ningún problema. Entre las más comunes y extendidas encontramos JAVA y PHP como lenguajes de desarrollo junto a MySQL y PosgreSQL como sistemas de gestión de bases de datos. Aún así, cualquier otra se puede integrar perfectamente en una aplicación RIA. En Edertone somos especialistas en las principales tecnologías de servidor, por lo que usted podrá escoger la que mejor se adapte a sus necesidades.

Tecnologías servidor

Actualmente existen muchas tecnologías que permiten la creación de aplicaciones RIA. Aún así, para el desarrollo en entornos profesionales sobresale la plataforma FLEX de Adobe, que ofrece un entorno multiplataforma y totalmente independiente del navegador o sistema operativo usados. Dicha tecnología se complementa perfectamente con el runtime AIR para la creación de aplicaciones de escritorio multi plataforma. Como alternativa open source existe la tecnología HTML5 o el framework GWT, que permite dotar de mayor interactividad a las aplicaciones web multi navegador. En Edertone le asesoramos para que escoja la tecnología que mejor encaje con los requerimientos de su proyecto y nos ocupamos íntegramente del desarrollo del mismo.
